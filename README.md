# File System Manager (fs_manager)
Консольная и графическая утилита для управления файловой системой. Разработана в качестве учебного проекта.
# Установка
1. Склонируйте репозиторий.
2. Убедитесь, что у вас установлен Python 3.8+
3. Установите зависимости и саму утилиту как пакет (находясь в корне проекта):
``` pip install -r requirements.txt```
```pip install -e .```


# Использование графического интерфейса (GUI)
Новая фича предоставляет современный оконный интерфейс с нативными пикерами файлов и обработкой ошибок.
После установки пакета запустить графическую версию можно командой:
```fs-manager-gui```


(Также доступен запуск скрипта напрямую: ```python fs_manager/gui.py```)
# Особенности GUI:
* Выбор инструмента из выпадающего списка.
* Поддержка нативных "файл-пикеров" (кнопки с иконками папки/файла рядом с текстовым полем).
* Встроенная консоль, которая перехватывает вывод результатов от основных скриптов и выводит их в окне программы.
* Полная отказоустойчивость: при неправильном вводе программа показывает сообщение (Snackbar) об ошибке красного цвета и не "вылетает".
* Интерактивные тултипы при наведении на интерфейсные элементы.
# Использование консольного интерфейса (CLI)
После установки также доступна классическая команда ```fs-manager```.
Для вызова справки используйте:
```fs-manager -h```


### Доступные команды:
* copy: Копирование файла.
```fs-manager copy source.txt dest.txt```
* delete: Удаление файла или папки.
```fs-manager delete folder_name```
* count: Подсчет файлов в папке.
```fs-manager count ./my_folder```
* search: Поиск файлов по регулярному выражению.
```fs-manager search ./my_folder "\.py$"```
* add-date: Добавление даты создания в название файла.
```fs-manager add-date ./my_folder --recursive```
* analyse: Анализ размеров файлов и папок.
```fs-manager analyse ./my_folder```
* backup (Дополнительная фича): Создание zip-архива файла или папки.
```fs-manager backup ./my_folder ./backup_name.zip```
